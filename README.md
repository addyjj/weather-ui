# Weather UI

This is a UI for displaying weather data from an Ambient weather station.

## Getting Started

1. Ensure you have [NodeJs v24.13.0+](https://nodejs.org/en/download) installed.

1. Install dependencies
   ```shell
   npm install
   ```
1. Run locally
   ```shell
   npm run dev
   ```

## Dependencies

- [React](https://react.dev) - Web component library.
- [Vite](https://vite.dev) - Fast frontend build tool.
- [Prettier](https://prettier.io) - Consistant formating.
- [EsLint](https://eslint.org) - Detects code issues.
- [Storybook](https://storybook.js.org) - Tool for building, testing, and documenting components in isolation.
